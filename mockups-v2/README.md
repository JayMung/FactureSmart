# mockups-v2/

**Ce dossier contient des maquettes HTML de design uniquement.**

Ces fichiers représentent les écrans de l'application FactureSmart sous forme de pages HTML statiques. Ils ont été utilisés lors de la phase de conception UI/UX pour valider les layouts, flux et composants.

## 📁 Contenu
- `DESIGN_PLAN.md` — Plan de conception global
- `DESIGN_SYSTEM.md` — Tokens de design system (couleurs, typographie, espacements)
- `screen-*.html` — Maquettes individuelles par écran
- `landing.html`, `maquettes.html`, `index.html` — Pages de présentation

## ⚠️ Notes
- Ces fichiers ne sont **pas utilisés dans le build de production**
- Ils ne sont pas inclus dans le bundle (`mockups-v2/` exclu du build Vite)
- Servent uniquement de référence pour le développement UI
- Les implémentations réelles sont dans `src/pages/` et `src/components/`
